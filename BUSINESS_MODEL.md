# 풋살 기록기 상용화 비즈니스 모델

## 과금 구조

### 팀(동호회) 단위 구독

| 플랜 | 가격(월) | 내용 |
|------|---------|------|
| Free | 0원 | 1팀, 기록 저장 30일, 광고 |
| Basic | 9,900원 | 1팀, 무제한 기록, 시즌 통계 |
| Pro | 19,900원 | 3팀, 다중 관리자, 데이터 내보내기 |
| Club | 39,900원 | 무제한 팀, API 연동, 브랜딩 커스텀 |

결제 주체: 동호회 총무/회장 (1명이 결제, 멤버는 무료 사용)

### 부가 기능 단건 결제

| 기능 | 가격 | 설명 |
|------|------|------|
| 시즌 리포트 PDF | 3,900원/회 | 시즌 종료 시 전체 통계 리포트 생성 |
| 선수 카드 이미지 | 1,900원/명 | 개인 기록 카드 (SNS 공유용) |
| 대회 모드 | 9,900원/회 | 토너먼트 브라켓, 대회 페이지 생성 |
| 영상 하이라이트 연동 | 4,900원/월 | 골 기록에 영상 클립 첨부 |
| 팀 추가 | 4,900원/팀/월 | Free/Basic 플랜에서 추가 팀 운영 |

### 구장/리그 B2B

| 대상 | 가격 | 가치 |
|------|------|------|
| 풋살장 | 월 49,900원~ | 자체 브랜딩 기록 시스템, 예약 연동, 고객 재방문 유도 |
| 지역 리그 | 월 99,900원~ | 리그 순위표, 팀 간 통계, 공식 기록 관리 |
| 대회 주최 | 건당 199,000원~ | 대회 전용 페이지, 실시간 스코어보드 |

## 상용화 시 기술 전환

| 현재 | 상용화 |
|------|--------|
| Google Sheets DB | PostgreSQL / Supabase / Firebase |
| Apps Script 백엔드 | Node.js / Next.js API |
| 단일 HTML 파일 | React + Vite 빌드 |
| 하드코딩 팀 1개 | 멀티테넌트 (팀별 분리) |
| localStorage 인증 | JWT + OAuth (카카오/네이버 로그인) |
| GitHub Pages | Vercel / AWS |

## MVP 전략

1단계: 현재 앱 무료 배포 → 풋살 동호회 커뮤니티에서 사용자 확보
2단계: 팀 구독 모델(Basic 9,900원) 도입 → 시즌 통계, 데이터 보관이 킬러 피처
3단계: 구장/리그 B2B 확장

핵심 수익원: 팀 추가 과금 + 구장 B2B
